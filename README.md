# Math Race Duel: Neon Circuit

**Math Race Duel: Neon Circuit** adalah game edukasi matematika berbasis web dengan konsep balapan neon cyber racing. Game ini dibuat agar latihan matematika terasa lebih seru, kompetitif, interaktif, dan cocok dimainkan oleh 2 pemain dalam satu perangkat.

##  Live Demo

Mainkan game di sini:

https://math-race-duel-neoncircuit.netlify.app/

##  Fitur Utama

* Game matematika 2 pemain
* Konsep neon cyber racing
* Mode tingkat soal SD, SMP, dan SMA
* Timer permainan
* Kalkulator jawaban
* Race progress dengan mobil futuristik
* Sistem skor, benar, salah, combo, dan akurasi
* Leaderboard tersimpan di browser
* Modal hasil akhir/pemenang
* Form kritik dan saran setelah game selesai
* Responsive untuk HP, tablet, laptop, touchscreen, dan TV

##  Cara Bermain

1. Buka link live demo.
2. Masukkan nama pemain.
3. Pilih mode permainan.
4. Klik mulai.
5. Setiap pemain menjawab soal matematika masing-masing.
6. Jawaban benar membuat mobil melaju.
7. Pemain yang mencapai finish atau bertahan sampai akhir menjadi pemenang.
8. Setelah game selesai, pemain bisa mengirim kritik dan saran.
